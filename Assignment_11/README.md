# Transaction CRUD API

A simple Spring Boot CRUD API built using Gradle.

## Features
- Create Transaction
- Update Transaction
- Delete Transaction
- Get All Transaction

## Tech Stack
- Java
- Spring Boot
- Gradle

## Run the Project
1. Clone the repository
2. Open in IntelliJ
3. Run the main application class

## API Testing
Postman collection is included in the `postman` folder.
